# PLAYWRITE WITH JS

This repository contains JavaScript examples and Playwright test scripts for learning browser automation and testing with Playwright.

## Files

- `chapter1basics.js` - Basic Playwright script examples.
- `test.js` - Additional test script.

## Usage

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run Playwright scripts with Node.js:
   ```bash
   node chapter1basics.js
   ```
